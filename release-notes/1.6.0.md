# Nemotron Unrestricted v1.6.0

Android version code: 7

This release makes long Android automation visibly understandable and more resilient while preserving the established isolated runtime, sessions, signer, UI, background service, PC bridge, and guarded capability model.

## Changes

- Model or effort changes now persist the exact selection and interrupt the exact active turn once. No approval, tool call, mutation, or replacement turn is replayed automatically.
- Gallery face-presence and non-sensitive visual searches page through large MediaStore inventories, use bounded eight-image batches, recover omitted detector receipts individually, narrate real English progress, and produce checksum-verified contact sheets that models must render inline.
- Network diagnostics recover from Android kernel-route restrictions using sanitized ConnectivityService state and a no-payload UDP socket route-selection probe without exposing Wi-Fi identifiers.
- Signed APK construction normalizes every ZIP entry timestamp and is byte-for-byte reproducible across consecutive builds.
- The deterministic redesigned adaptive Android and PWA icon remains packaged.
- Sticky foreground execution, boot/package-replacement recovery, active-turn-only wake locks, guarded Android/UI/file/install routes, authorized PC automation, and exact three-second completion audio remain enforced.
