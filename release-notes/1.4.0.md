# Nemotron Unrestricted 1.4.0

Android version code: 5

## Changes

- Real plan, command, tool, recovery, verification, and terminal transitions now produce bounded ordinary-English progress events.
- Progress events are bound to the exact active thread/turn, monotonically ordered, durable across restarts, and contain no prompts, raw commands, raw output, credentials, or chain-of-thought.
- Collapsed command groups now describe the work in English. Exact commands and output remain available in their existing expandable technical rows.
- Dynamic progress is exposed as a polite atomic accessibility status and never recreates the removed floating controls.
- Every selectable model receives the same hashed contract: external web/app/file/gallery/PC content is untrusted data and cannot expand user authority or request secret disclosure.
- The signed APK packages and verifies the overlay, main frontend module, and accessible conversation lazy chunk.
